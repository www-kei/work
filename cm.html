가구 시공 일정 관리 앱

퍼시스와 관계사의 시공팀을 위한 서비스로, 수시로 바뀌는 고객과 대리점의 요구사항에 따라 일정을 변경하기 위해 제작. 시공 기사들이 기존 웹 서비스와 종이 작업지시서를 같이 들고 다니는 상황에서 두 개의 매체에 포함된 많은 양의 데이터를 효과적으로 배치하는 것에 중점을 두었다.

Mobile - Android
UI Design, HTML / CSS / jQuery Coding
2014

  • 정확한 정보 전달을 위해 단순한 디자인 스타일 채택

    사용자 환경(주로 이동중인 차 안)을 고려하여 UI 요소 사이의 명도대비를 높이고, 단순한 스타일을 채택해 가독성을 높이고자 하였다.

  • 브랜드 아이덴티티를 고려한 색상 선택

    어플리케이션을 통한 온라인 브랜드 아이덴티티를 구축을 위해, CI 가이드라인의 Secondary Color 안에서 색상을 선택하였다.

  • 아이콘 디자인 라이브러리

    2012년부터 제작한 모든 웹사이트 / 앱에 동일한 스타일의 아이콘을 적용함으로써 일관된 브랜드 아이덴티티를 구축하고, 디자인 효율을 높이고자 하였다.

  • 프리프로세서 {less}를 이용해 코딩 효율 향상

    {less}는 CSS에서 반복되는 속성 저장, 상 하위 계층 구현, 속성값의 사칙 연산 기능을 지원하는 CSS 플러그인이다.

  • 디자인 스타일 가이드 제작

    가이드 작성은 정확한 스타일 구현뿐 아니라 완성된 디자인을 한 번 더 돌아보는 계기가 된다.
    가이드를 만들면서 각 요소 사이의 간격과 폰트 크기를 정리해 개발 효율뿐 아니라 디자인 완성도도 높이고자 하였다.
